# BangumiBar v1.1.1

## 版本简介

本版本是一次围绕“在看体验”做的小版本修正发布，重点是统一完结补评流程、移除不再需要的旧交互，并整理遗留代码与发布说明。

## 主要更新

- 调整分集进度交互，移除此前的多选与批量标记逻辑，回归更直接的单集点击标记
- 修复点完最后一集后未自动弹出窗口的问题，改为在正片全部完成时直接打开“修改条目”弹窗
- 统一完结补评入口，使用“修改条目”弹窗完成 `看过`、评分与吐槽，不再保留旧评分专用弹窗
- 清理旧评分弹窗组件、批量标记相关状态、样式与无入口逻辑，减少维护负担
- 更新 README 中过时的功能说明，使项目说明与当前行为保持一致

## 发布资产

- `BangumiBar-1.1.1-mac-arm64.zip`

## 已知说明

- 当前 macOS 版本仍为未签名版本
- 首次打开时，系统可能提示无法验证开发者，需要在 macOS 中手动放行
- Windows 与 Linux 打包入口仍然保留，但本次发布资产以当前机器生成的 macOS arm64 版本为主
